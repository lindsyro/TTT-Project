# Проект Крестики-Нолики

## Project Backend TypeScript

Создаем веб-приложение на языке **TypeScript** с использованием среды `Node.js` и фреймворка `Nest.js` с поддержкой базы данных и авторизацией.
         
### Темы:
- Node.js;
- Nest.js;
- API;
- Паттерн MVC;
- Авторизация;
- Объектно-реляционное отображение.

-----

## Project Frontend TypeScript

Создаем клиентское приложение с использованием фреймворка `Vuejs` для реализованного сервера.

### Темы:
- Фрейморк `VueJS`;
- Авторизация;
- Объектно-реляционное отображение;
- Паттерн `MVVM.`

### Приложение поддерживает:
- авторизацию по паре логина и пароля (basic auth);
- возможность регистрации;
- отображение списка текущих игр;
- возможность создания игры;
- возможность присоединения к игре;
- отображение информации об участниках;
- возможность играть в крестики-нолики;
- сохранение полученной информации в базу данных;
- выход на экран авторизации.

Используются библиотеки `axios` и `vuex`.
